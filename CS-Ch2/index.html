无符号数的减法

  1. 获取被减数和减数的值。
  2. 如果减数大于被减数，则将减数与模数取模，以确保结果在模数范围内。这是为了模拟无符号数的循环性质。
  3. 执行模数减法，即将被减数减去减数。
  4. 如果差值小于 0，则将差值与模数相加，以确保结果在模数范围内。这是为了处理模数减法产生的负值。
  5. 最终得到的差值即为无符号数的减法结果。

例如：考虑一个 8 位无符号数的减法，模数为 256（2^8）。

假设被减数为 50，减数为 70。

  1. 由于减数 70 大于被减数 50，对减数进行模数操作，得到减数 70 % 256 = 70。
  2. 执行模数减法：差值 = 被减数 - 减数 = 50 - 70 = -20。
  3. 由于差值 -20 小于 0，将其与模数相加，得到最终结果：差值 = -20 + 256 = 236。
