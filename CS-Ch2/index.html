Bool代数

  • C语言中的位级运算：&， |， ~， ^

    • 适用于任何整型数据类型

    • 将操作数视为位向量

    • 将操作数按位运算

    • ``` c
      // 异或操作的示例
      void inplace_swap(int *x, int *y){
      *x = x^y;
      *y = y^x;
      *x = x^y;
      }

  • 有符号数（补码）

    • 真实的结果需要w+1位，丢弃最高有效位，剩余部分视为补码的到最终结果
    • 正溢出 z=x+y-2^w
    • 正常 z=x+y
    • 负溢出 z=x+y+2^w
    • 如何判断是否溢出?
      • Xh!=Yh或Xh=Yh=Zh则无溢出
      • Xh=Yh!=Zh则溢出
